Parkwood Vale Harriers

This is a training log application for my CG2 Computing coursework. It is built using Flask, Jinja2 and JavaScript, with a number of other open source projects, such as Bootstrap, also included. It's easy to get started with it - just download the files, and run the following:

$ virtualenv env
$ source env/bin/activate
$ pip install -r requirements.txt
$ python manange.py runserver

It's still in development, and very little actually works yet (read: nothing), but it's coming along nicely, and should be feature complete by Aprilish.
